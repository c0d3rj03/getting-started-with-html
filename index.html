HTML

It was invented in the 1990's.
This stands for Hypertext Markup Language.
It is what browsers interpret when showing web pages.

Tags

This is part of the hidden language.
Learned various types of tags such as:
- b for bold
- em for italics

The "a" tag is an attribute tag.

Right now, we learned how to give a hyperlink as part of an attribute (using href). Such as:
My Weather Data on Weather Underground

Most tags have content.

Newlines

Just pressing enter to start a new line in an HTML file will not cause a newline to appear. Must use a tag to cause a newline to be created by the browser. A few different ways learned:
- "br" will cause a newline break
- p and /p will also do that.

Void tags

As noticed with b and /b and others, most tags need a corresponding end tag. If that doesn't appear, then the tag interpreted by the browser will continue in that manner.

Void tags are tags that have no content, thus do not need an end tag. "br" is an example because a "/br" isn't needed.

Inline versus Block

Block elements will actually create an invisible box that can be formatted with additional attributes.

Inline elements simply will just cause a character entry.

Examples of block are p and div.

Also, learned that
- strong tag works like b.
- form seems like a block element like p and div.
